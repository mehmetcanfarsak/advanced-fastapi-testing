## Installation

```bash
git clone https://github.com/mehmetcanfarsak/advanced-fastapi-testing.git
cd advanced-fastapi-testing
pip install -r requirements.txt
```

## How to run the app

```bash
uvicorn main:app --reload
```

```
http://localhost:8000/docs
```

